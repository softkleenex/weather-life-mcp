## 5. 빨래지수 (v2.0)

### 빨래하기 좋은 날인지 확인
```
"오늘 빨래해도 돼?"
"빨래 널어도 될까?"
"빨래하기 좋은 날이야?"
```

### 응답 예시
```json
{
  "location": "서울",
  "laundry_score": 85,
  "grade": "최적",
  "is_good": true,
  "factors": {
    "humidity": {"value": 45, "score": 90},
    "precipitation": {"probability": 10, "score": 90},
    "wind": {"speed": 3.0, "score": 80}
  },
  "recommendation": "빨래하기 좋은 날! 오후 3시 전에 널어두세요.",
  "best_time": "10:00-15:00"
}
```

### 빨래지수 해석

| 점수 | 등급 | 의미 |
|------|------|------|
| 80-100 | 최적 | 빨래 건조 최적, 걱정 없이 널어도 OK |
| 60-79 | 좋음 | 건조 잘 됨, 오후 전 걷기 권장 |
| 40-59 | 보통 | 건조에 시간 걸림, 실내 건조 고려 |
| 0-39 | 나쁨 | 실내 건조 권장 |

## 9. 김장지수 (v2.0) - 세계 유일!

### 김장하기 좋은 날인지 확인 (11-12월 한정)
```
"김장 언제 하면 좋아?"
"오늘 김장해도 돼?"
"김장하기 좋은 날씨야?"
```

### 응답 예시
```json
{
  "location": "서울",
  "kimjang_score": 90,
  "grade": "최적",
  "is_good": true,
  "is_season": true,
  "factors": {
    "temperature": {"min": -2, "max": 5, "score": 95}
  },
  "recommendation": "김장 적기! 배추 절이기 최적 온도입니다.",
  "tips": [
    "배추 절임: 새벽~오전 (가장 추운 시간)",
    "양념 만들기: 실내에서",
    "버무리기: 오후 (손 시리지 않게)"
  ],
  "folklore": "첫눈 오기 전 김장이 맛있다는 속설이 있어요!"
}
```

### 김장지수 기준

| 기온 | 등급 | 설명 |
|------|------|------|
| -3 ~ 4°C | 최적 | 김장 황금 온도 |
| 5 ~ 8°C | 좋음 | 김장 가능 |
| 9°C+ | 별로 | 너무 따뜻함 |
| -5°C 이하 | 나쁨 | 너무 추움 |

## 13. 복합 질문

AI가 여러 Tool을 조합하여 답변할 수 있습니다.

```
"서울 날씨랑 미세먼지 같이 알려줘"
→ get_weather + get_air_quality_info

"오늘 뭐하면 좋을까? 옷은 뭘 입지?"
→ what_should_i_do_today + get_outfit_recommendation

"주말에 등산 갈까 피크닉 갈까?"
→ is_good_for_hiking + is_good_for_picnic

"빨래하고 세차도 할까?"
→ is_good_for_laundry + is_good_for_car_wash
```

## Tips

1. **자연어로 편하게** - 형식에 구애받지 않고 질문
2. **종합 추천 활용** - "오늘 뭐하면 좋을까?"로 한 번에 확인
3. **한국 특화 기능** - 빨래, 등산, 피크닉, 세차, 김장 지수
4. **80개+ 지역 지원** - 서울 25개 구, 경기도, 광역시, 주요 도시
5. **실시간 데이터** - 기상청/에어코리아 실시간 연동
