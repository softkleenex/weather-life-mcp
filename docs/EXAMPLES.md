## 5. 빨래지수 (v2.0)

### 빨래하기 좋은 날인지 확인
```
"오늘 빨래해도 돼?"
"빨래 널어도 될까?"
"빨래하기 좋은 날이야?"
```

### 응답 예시
```json
{
  "location": "서울",
  "laundry_score": 85,
  "grade": "최적",
  "is_good": true,
  "factors": {
    "humidity": {"value": 45, "score": 90},
    "precipitation": {"probability": 10, "score": 90},
    "wind": {"speed": 3.0, "score": 80}
  },
  "recommendation": "빨래하기 좋은 날! 오후 3시 전에 널어두세요.",
  "best_time": "10:00-15:00"
}
```

### 빨래지수 해석

| 점수 | 등급 | 의미 |
|------|------|------|
| 80-100 | 최적 | 빨래 건조 최적, 걱정 없이 널어도 OK |
| 60-79 | 좋음 | 건조 잘 됨, 오후 전 걷기 권장 |
| 40-59 | 보통 | 건조에 시간 걸림, 실내 건조 고려 |
| 0-39 | 나쁨 | 실내 건조 권장 |

## 15. 복합 질문

AI가 여러 Tool을 조합하여 답변할 수 있습니다.

```
"서울 날씨랑 미세먼지 같이 알려줘"
→ get_weather + get_air_quality_info

"등산하기 좋으면 근처 맛집도 알려줘"
→ is_good_for_hiking + search_restaurant

"주말에 등산 갈까 피크닉 갈까?"
→ is_good_for_hiking + is_good_for_picnic

"홍대 데이트 코스랑 길찾기까지"
```

## Tips

1. **자연어로 편하게** - 형식에 구애받지 않고 질문
2. **장소 검색 활용** - "강남역 근처 카페"로 즉시 검색
3. **스마트 코스** - "데이트 코스 짜줘"로 날씨 맞춤 코스
4. **한국 특화 기능** - 빨래, 등산, 피크닉, 세차 지수
5. **80개+ 지역 지원** - 서울 25개 구, 경기도, 광역시, 주요 도시
6. **실시간 데이터** - 기상청/에어코리아/카카오맵 실시간 연동
