## 심사 기준 대응 (v3.7)

### 창의성 - 네이버 날씨/기상청 앱과 뭐가 다른가?

| 기존 서비스 | Weather Life MCP |
|-------------|------------------|
| 대시보드형 정보 나열 | **대화형 의사결정 지원** |
| 날씨 → 사용자가 판단 | **날씨 → AI가 행동 추천** |
| 단일 서비스 | **날씨+장소+길찾기 통합** |
| 앱/웹 고정 | **모든 AI 플랫폼에서 동작** |
| 생활지수만 제공 | **A→B→C 코스 + 거리 계산** |

**진짜 차별점:**
- "강남에서 친구랑 뭐하지?" 한 마디 → 날씨 체크 → 실내/야외 판단 → 3단계 코스 + 장소 간 거리 + 카카오맵 링크
- 이 **"질문→판단→행동→이동"** 파이프라인은 기존 서비스에 없음

### 편의성 - 한 마디로 코스 완성!
- 자연어로 간단하게 요청 가능
- "강남에서 친구랑 뭐하지?" → 날씨 맞춤 A→B→C 코스 추천
- **30개 Tool**로 다양한 상황 대응
- 카카오맵 원클릭 길찾기 링크
- 가까운 역 기준 거리 안내 (예: "강남역에서 도보 837m")
- Claude, ChatGPT, Gemini 등 **모든 AI에서 동일하게 동작**

### 안정성 - 3사 API + 표준 프로토콜
- 기상청 + 에어코리아 + 카카오맵 API 연동
- **MCP 1.0 표준 프로토콜** (JSON-RPC 2.0)
- Railway 배포로 24시간 가동
- Stateless HTTP로 확장성 확보
- 에러 처리 및 폴백 로직 구현
