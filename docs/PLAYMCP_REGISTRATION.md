### 상세 설명 (v2.5)
```
Weather Life는 한국인의 생활 패턴에 맞춘 날씨 기반 MCP 서버입니다.

[기본 기능]
- 실시간 날씨와 미세먼지 조회
- 기온별 옷차림 추천
- 외출 적합도 점수 (0-100)

[한국 특화 기능]
- 빨래지수 - 빨래 건조하기 좋은 날인지 판단
- 등산지수 - 등산 적합도 + 추천 산 안내
- 피크닉지수 - 한강/공원 가기 좋은 날 + 치맥 타임
- 세차지수 - 내일 비 여부까지 고려
- 김장지수 - 11-12월 김장 적기 판단 (세계 유일!)

[v2.3 건강지수]
- 감기위험지수 - MIT, Yale, PNAS 연구 기반
- 출퇴근지수 - 교통수단별 점수
- 알레르기지수 - 꽃가루, 황사 연동

[v2.4 라이프스타일지수]
- 편두통지수 - 기압/습도 연구 기반
- 수면지수 - 최적 온습도 분석
- 사진지수 - 골든아워 계산
- 관절통지수 - 관절염 연구 기반

[v2.5 레저/스포츠지수] NEW!
- 드라이브지수 - 시야/노면/결빙 분석
- 캠핑지수 - 낙뢰/강풍 위험도
- 낚시지수 - 기압변화 기반 분석
- 골프지수 - 바람/볼비행 영향
- 러닝지수 - 열지수/AQI 통합
- 바베큐지수 - 화재안전/바람 분석

- 서울 25개 구, 경기도, 광역시 등 80개+ 지역 지원

"오늘 뭐하면 좋을까?" 한 마디로 모든 정보를 받아보세요!
```

## 심사 기준 대응

- **빨래지수** - 기상청 종료 서비스 부활

- **골프지수** - 바람/볼비행 영향 분석 (v2.5)
- **러닝지수** - 열지수/AQI 통합 건강 분석 (v2.5)

## 사용 시나리오

### 시나리오 1: 주말 활동 고민
```
사용자: "오늘 뭐하면 좋을까?"

AI: 오늘은 등산하기 좋아요! (85점)

활동별 점수:
- 등산: 85점 (최적)
- 피크닉: 70점 (좋음)
- 빨래: 60점 (보통)
- 세차: 45점 (별로)
```

### 시나리오 2: 빨래 고민
```
사용자: "오늘 빨래해도 돼?"

AI: 빨래하기 좋은 날이에요! (82점)

- 습도 45%, 바람 3m/s로 건조 최적
- 강수확률 10%
- 오후 3시 전에 걷는 것 추천!
```

### 시나리오 3: 김장 시즌
```
사용자: "김장 언제 하면 좋아?"

AI: 이번 주말이 김장 적기예요! (90점)

- 토요일: 최저 -1C, 최고 8C
- 배추 절이기 최적 온도
- 일요일 오후 눈 예보, 토요일 추천!
```
