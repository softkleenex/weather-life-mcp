- `get_smart_course` - 날씨 연동 A→B→C 코스 추천 (장소 간 실제 거리 계산)

- `search_nearby_places` - 전국 장소 검색
- `search_restaurant` - 맛집 검색
- `get_directions_link` - 카카오맵 길찾기

- `get_place_recommendation` - 상황별 맞춤 추천 + 가까운 지하철역/버스터미널 기준 거리
