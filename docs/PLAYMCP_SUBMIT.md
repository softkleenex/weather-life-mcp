## v3.6.0 주요 기능

### 스마트 코스 (1개)
- `get_smart_course` - 날씨 연동 A→B→C 코스 추천 + 하루 일정 계획

### 장소 검색 (5개)
- `search_nearby_places` - 전국 장소 검색
- `search_restaurant` - 맛집 검색
- `get_directions_link` - 카카오맵 길찾기
- `get_recommended_spots` - 활동별 장소 추천
- `get_place_recommendation` - 상황별 맞춤 추천 (추천 이유, 이동 방법, 알아야 할 것 제공)

### 날씨/생활 지수 (22개)
- 날씨, 미세먼지, 옷차림(TPO별), 외출적합도, 자외선, 식중독
- 빨래/등산/피크닉/세차/운동지수
- 캠핑/낚시/골프지수
- 감기위험/편두통/관절통/수면/사진촬영/알레르기/출퇴근지수
